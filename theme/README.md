# The Liège Data Hub — Ghost theme V2

Anthropic-inspired editorial Ghost theme using the original The Liège Data Hub logo and floppy-disk mascot.

## Install

1. Ghost Admin → Settings → Design.
2. Upload the original logo/mascot as the Site logo.
3. Change theme → Upload theme.
4. Zip the contents of this `theme` directory and upload the ZIP.
5. Configure the homepage fields under Settings → Design → Site-wide.
6. Configure navigation under Settings → Navigation.

## Recommended Ghost settings

- Hero title: `Make it happen.`
- Hero subtitle: `Turning data strategy into platforms that deliver.`
- Hero description: `Frameworks, architecture and pragmatic execution for organisations that want data initiatives to create measurable value.`
- Primary CTA: `Explore the frameworks`
- Secondary CTA: `Read the latest thinking`
